# LCPM Paper 1: Code and Data

Companion repository for "Luoshu Chiral Potts Model: Ground State Perfection and First-Order Phase Transition"

## Scripts

- `hlcpm_ground_state_enumeration.py`: Exhaustive enumeration of 5^9 configurations
- `ypm_phase_diagram.py`: Critical point and order parameter calculations
- `z5_aon_verify.py`: All-or-nothing theorems verification
- `climate_z5_verify.py`: CUG-CMA CHDT data processing, Z5-Z6 effect size analysis, boundary jump tests
- `generate_figures.py`: Generation of all six manuscript figures

## Data

- `cug_cma_gridyear_full.csv`: CUG-CMA CHDT processed grid-year data (1880-2020, 186 land grid points)

## Figures

Six manuscript figures (Figure 1-6) in PNG format.
